# 🛒 Cartify - Shopping Cart App

🚀 Live Demo: https://cartify4u.netlify.app/

## 📌 About the Project

Cartify is a modern and responsive shopping cart web application built using **React.js and Vite**. It allows users to browse products, add items to cart, manage quantities, and view total price dynamically. The project uses clean UI design and smooth navigation for a seamless shopping experience.

---

## ✨ Features

- 🛍️ Product listing page  
- 🛒 Add to cart / remove from cart  
- 🔢 Dynamic quantity update  
- 💰 Real-time total price calculation  
- 🔔 Toast notifications for user actions  
- 🌐 React Router navigation  
- ⚡ Fast and optimized with Vite  

---

## 🛠️ Tech Stack

- React.js  
- Vite  
- React Router DOM  
- Context API (State Management)  
- React Toastify  
- Lucide React Icons  
- CSS / Tailwind (if used)
